# 💼 Personal Portfolio Website

A modern, responsive personal portfolio website developed using **HTML, CSS, and JavaScript**. This project showcases my profile, skills, education, academic projects, and contact information with a clean and interactive user interface.

## 📌 Features

- 📱 Fully responsive design
- 🏠 Hero section with introduction
- 👤 About Me section
- 💻 Skills section
- 📂 Projects showcase
- 🎓 Education timeline
- 📧 Contact section with contact form
- 🌐 Social media links
- 🍔 Responsive navigation menu
- ✨ CSS animations and hover effects

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons

## 🚀 How to Run

1. Download or clone this repository.

```bash
git clone https://github.com/Seha30/Portfolio.git
```

2. Open the project folder.

3. Double-click **index.html** or open it using your preferred web browser.

No additional software or installation is required.
